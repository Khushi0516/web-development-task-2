# Interactive Calculator Project

## ✨ Overview

This is an interactive web-based calculator project developed as **Level 1 - Task 2** for the Cognifyz Internship program. The project showcases fundamental JavaScript concepts through three engaging interactive features in a beautifully designed interface.

## 🎯 Features

### 1. 🎨 Color Change Button
- Interactive button that changes color and text on click
- Toggles between blue and red backgrounds
- Smooth hover animations

### 2. ⏰ Time-Based Greeting
- Displays personalized greetings based on current time
- Shows alerts and updates the page with time-appropriate messages:
  - Good Morning (before 12 PM) ☀️
  - Good Afternoon (12 PM - 5 PM) 🌤️
  - Good Evening (5 PM - 9 PM) 🌇
  - Good Night (after 9 PM) 🌙

### 3. 🧮 Smart Calculator
- Basic arithmetic operations: Addition (+), Subtraction (-), Multiplication (×), Division (÷)
- Input validation for numbers
- Error handling for division by zero
- Clear function to reset inputs and results
- Real-time result display

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with glassmorphism effects, animations, and responsive design
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation

## 🎨 Design Features

- **Glassmorphism UI**: Semi-transparent cards with backdrop blur effects
- **Animated Gradient Background**: Dynamic color-changing background
- **Responsive Design**: Mobile-friendly layout
- **Smooth Animations**: Hover effects and transitions
- **Modern Typography**: Clean, readable fonts

## 🚀 How to Run

1. **Clone or Download** the project files
2. **Open** `index.html` in any modern web browser
3. **Interact** with the three features:
   - Click the color change button
   - Try the greeting feature
   - Use the calculator for basic math operations
